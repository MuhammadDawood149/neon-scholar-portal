## Features

### Admin
- Manage users (Add/Edit/Delete)  
- Assign courses and teachers  
- Assign parents to students  
- Generate attendance and result reports  

### Teacher
- View assigned courses  
- Mark and update attendance  
- Upload quiz, assignment, midterm, and final marks  
- Generate performance reports  

### Student
- View attendance and enrolled courses  
- View results with grade calculation  
- Download transcript  

### Parent
- View child’s attendance and results
